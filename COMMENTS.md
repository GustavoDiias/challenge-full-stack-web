Execução do Projeto:
    -Back End:
        yarn install
        yarn sequelize db:migrate
        yarn start

    -Front End
        yarn install
        yarn serve

Decisão da arquitetura utilizada:
    Arquitetura MVC
    Front End: Vuetifyjs como framework de UI
    API: Express e NodeJS
    Banco de Dados: Postgress
    Idioma de escrita do código: Inglês
